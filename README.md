车冲检测管理平台项目

## 安装依赖

`npm install`

## 启动服务

`npm run dev`

## 登录

- 账号：admin 密码：随便输
- 账号：editor 密码：随便输

## 注意

- 开发环境服务：192.169.4.22:8010
- 部署环境服务：192.169.5.45:8010

- 为了方便管理即部署，在所有的接口钱统一加了`api/`，
  - 本地开发时，若连开发环境服务 ,在代理的时候需要把接口请求的`api/`替换为空；
  - 本地开发时，若连部署环境服务，代理的时候不需要`rewrite`，但是需要给图片也代理一下（解注释）;
  - 打包部署时, 代理的时候需要把`api/`替换为空
